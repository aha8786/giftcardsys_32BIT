너는 Python GUI 개발 전문가다.
(Tkinter 또는 PyQt 사용 가능, 기본은 Tkinter로 구현)

나는 POS 환경에서 사용할 프로그램을 개발하고 있다.

---

# 🎯 목표

전체화면으로 실행되는 프로그램에서  
“최소화 버튼을 누르면 완전히 숨겨지고,  
대신 화면 위에 작은 복귀 버튼이 떠서  
언제든 다시 전체 화면으로 복귀할 수 있는 기능”을 구현하라.

---

# 📌 필수 기능

## 1️⃣ 최소화 동작 (핵심)

- 기존 최소화 버튼 또는 “숨기기 버튼” 클릭 시:
  - 메인 윈도우는 완전히 숨김 (withdraw)
  - 작업표시줄에서도 보이지 않게 처리

👉 사용자 입장에서는 “최소화된 느낌”이어야 한다

---

## 2️⃣ 항상 위 복귀 버튼 (핵심 기능)

- 메인 윈도우가 숨겨지면:
  - 작은 창(버튼)을 생성
  - 화면 한쪽 구석에 위치 (우측 하단 추천)

### 요구사항
- 항상 위에 표시 (`topmost = True`)
- 다른 프로그램 위에서도 보이도록
- 클릭 시:
  → 메인 윈도우 다시 표시 (deiconify)
  → 복귀 버튼은 사라짐

---

## 3️⃣ UX 요구사항 (중요)

- 복귀 버튼은 매우 직관적이어야 한다
  - 텍스트: “복귀” 또는 “열기”
- 크기:
  - 작지만 클릭 가능 (예: 80x40)
- 위치:
  - 화면 우측 하단 고정

---

## 4️⃣ 추가 기능 (반드시 포함)

### ✔ 단축키 복귀
- 예: Ctrl + Alt + A
- 어디서든 누르면 메인 화면 복귀

---

## 5️⃣ 안정성 요구

- 프로그램 종료 시:
  - 복귀 버튼도 같이 종료
- 중복 생성 방지:
  - 복귀 버튼이 여러 개 생성되지 않도록 처리

---
